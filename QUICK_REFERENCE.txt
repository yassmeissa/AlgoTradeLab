A: Check if port 8000 is free: lsof -i :8000
   Or use different port: uvicorn main:app --port 8001

Q: Swagger UI not loading?
A: Verify server is running: curl http://localhost:8000/docs
   Check for errors: python run.py

Q: Tests failing?
A: Install all dependencies: pip install -r requirements.txt
   Run: pytest -v for verbose output

Q: Database error?
A: Check PostgreSQL is running
   Verify DATABASE_URL in .env
   Check connection: psql -U user -d algotrade_db

Q: How to get API token?
A: Register: POST /api/auth/register
   Login: POST /api/auth/login
   Copy token from response
   In Swagger: Click Authorize, paste "Bearer TOKEN"
